# Flow 06 — Habits / Reference Tab

## Overview
The Habits tab is the user's personal health reference notebook.
It has four sections: daily non-negotiables (interactive,
resets each day), morning drink recipe (reference + habit log),
foods to avoid (reference only), and a full plan viewer.

The only interactive elements are the 5 acknowledgement cards
and the "Mark done" button on the recipe. Everything else is
read-only reference content pulled from planStore.

**Navigator:** `HabitsStack` inside `MainTabs`
**Stores:** `usePlanStore` · `useDailyLogStore` · `useUserStore`

## MorningDrinkScreen

```tsx
// screens/main/MorningDrinkScreen.tsx
// Data from plan.habits.morningDrink — all AI-generated, not hardcoded

const drink = usePlanStore(s => s.plan.habits.morningDrink);

// Consecutive days streak for this habit:
const drinkStreak = useDailyLogStore(s =>
  s.history.filter(d =>
    d.habits.find(h => h.habitKey === 'morningDrink' && h.status === 'done')
  ).length  // crude streak — improve later
);

// Mark done button:
const isDoneToday = useDailyLogStore(s =>
  s.today.habits.find(h =>
    h.habitKey === 'morningDrink' && h.status === 'done'
  )
);

const handleMarkDone = () => {
  useDailyLogStore.getState().logHabit({
    habitKey: 'morningDrink',
    status: 'done',
    loggedAt: new Date().toISOString(),
  });
  Haptics.notificationAsync(Haptics.NotificationFeedbackType.Success);
};
```

## Key UX rules

1. Non-negotiables come from **plan.habits.nonNegotiables** — never hardcoded
2. Acknowledgement is **one-way** (cannot un-acknowledge, except long-press same day)
3. Acknowledged rules contribute to the **habits component of dayScore**
4. Morning drink recipe comes from **plan.habits.morningDrink** — all AI-generated
5. Foods to avoid come from **plan.habits.avoidList** — AI-generated, not hardcoded
6. The plan viewer shows **today's routine** (weekday or weekend based on current day)
7. "Regenerate plan" button navigates to **Settings → Regenerate** — does not regenerate inline
8. All 5 non-negotiables reset at **midnight** via archiveToday()
9. The success state (all 5 done) shows a **trophy celebration** — subtle, not modal
10. Morning drink streak counter uses **consecutive done days** from history
